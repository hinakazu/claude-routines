**① Anthropic、評価額9,650億ドルでIPO申請――生成AI企業初の超大型上場へ**
AI安全性研究企業のAnthropicが6月1日、SECに機密S-1書類を提出し株式公開に向けた手続きを開始した。評価額は約9,650億ドルと、OpenAI（約8,520億ドル）を上回り民間AI企業として史上最大規模。幹事証券はMorgan StanleyとGoldman Sachsで、年間売上高は470億ドル、2028年の損益分岐点達成を見込む。上場は早ければ2026年10月と予測されている。
📜 歴史的背景: AnthropicはOpenAIの「商業化優先」路線に反発したDario・Daniela Amodei兄妹が2021年に創業。2022年末のChatGPT登場による生成AIブームで急成長し、Amazonなどから累計数百億ドルの出資を受けた後、ついに公開市場へ打って出る段階に達した。

**② 日経平均、史上最高値68,402円を更新――AI・半導体株がバブル超えを牽引**
2026年6月3日、東京株式市場で日経平均株価が前日比1,667円89銭高の68,402円13銭で引け、史上最高値を更新した。取引時間中は一時68,786円まで急騰し、TOPIXも4,000の大台を一時突破。AI・半導体関連銘柄への資金集中が指数全体を押し上げた。ただし6月5日には一転して5.38%安の63,838円まで急落し、市場の高ボラティリティも鮮明になった。
📜 歴史的背景: 日経平均はバブル絶頂期の1989年12月に38,915円を記録後、2009年3月に7,054円まで崩壊し「失われた30年」を経験。2024年2月に34年ぶりのバブル超えを達成し、円安・AI投資ブームを追い風に連続最高値更新を続けている。

**③ イラン、停戦合意後にイスラエルへミサイル発射――中東紛争が再燃**
2026年2月28日に米・イスラエルによるイラン攻撃で始まった「2026年イラン戦争」は4月7日に一時停戦合意に至ったが、イスラエルがレバノンのヒズボラ拠点を攻撃したことを受け、イランがイスラエルに向けミサイルを発射した。ホルムズ海峡の事実上の封鎖が継続中で、WTI原油価格は約5.7%急騰し供給不安がリスクプレミアムとして市場に織り込まれている。
📜 歴史的背景: イランは2002年以降の核開発問題で欧米と対立し、2015年のJCPOA（核合意）で一時緩和されたものの2018年にトランプ政権が離脱し制裁を再開。数十年にわたる核・制裁・プロキシ戦争の積み重ねが2026年の本格的軍事衝突へと発展した。

**④ Microsoft Build 2026：WindowsをAIエージェント専用プラットフォームへ全面刷新**
サンフランシスコで開催のMicrosoft Build 2026でSatya Nadella CEOが「Windowsはもはや人間だけのプラットフォームではない」と宣言し、Windows Agent Framework・Azure Agent Mesh・Project Solara（Qualcommと共同開発のエージェントファーストOS）を発表した。GitHub Copilotは8月にGPT-4から自社製AIへ自動移行し、NVIDIAはComputex 2026でRTX SparkスーパーチップをWindows PC搭載で秋リリースすると発表した。

**⑤ 世界海洋デー2026：「海との新たな関係」を国際社会が模索**
6月8日の世界海洋デーは「私たちが知る世界を超えて、海との新しい関係を描く」をテーマに開催。海洋は過剰漁業・プラスチック汚染・気候変動の三重苦にさらされ、ブルーエコノミー（持続可能な海洋経済）への転換が急務とされている。6月5日の世界環境デーはアゼルバイジャン主催で「#NowForClimate」を掲げ、150カ国超が参加した。
📜 歴史的背景: 世界環境デーは1972年のストックホルム国連人間環境会議を機に制定。2015年パリ協定・2023年「BBNJ条約」（公海条約）締結で国際的な海洋保護の枠組みが強化され、公海の30%保護目標が掲げられたことで、海洋と人類の関係の再定義が始まっている。
